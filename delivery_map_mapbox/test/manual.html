毎日の基本の流れ

1. アプリを開く 前回終了した場所の地図がすぐに表示されます。GPSを取得すると現在地へ移動します。位置情報の許可が出たら許可してください。
2. モードを確認する 左上付近の モード: 配送 / モード: 集荷 を確認します。メニューから切り替えできます。
3. 配送先・集荷先を準備する 配送は伝票撮影・音声・手入力で追加します。伝票が多い日は連続読み込みが便利です（配送先の追加参照）。集荷はメニューの 集荷コースを選択 を使います。
4. ❗要確認ピンをチェックする 赤い枠のピンは位置が怪しい可能性があります。出発前に ✎ で開いて確認・修正してください。
5. 順番を整える 車で配る時は「道路優先配送順（おすすめ）」を選ぶと、最初に配りたい町名とエリア順を相談できます。大森東を終えてから西糀谷へ進む、といった荷物のまとまりに合わせた順番にできます。件数が少ない日も同じ機能を使います。
6. 終わったら完了にする リストの ✓ を押すと完了です。通信が悪い場所で押した場合も、未同期として保存され、通信回復後に自動で再送されます。

配送モードと集荷モード

配送モード 単発の配送先を追加して回るためのモードです。配送先はこの端末のブラウザに保存されます。
集荷モード Googleスプレッドシートの集荷リストを取り込んで使うモードです。完了状態はスプレッドシートにも書き戻され、他の人の端末にも同期されます。

切り替えは、画面左上付近の モード ボタン、またはメニューから行います。

入力モードと配送モード

配送リストの見出しにある 入力 / 配送 で、その時の作業に必要な画面へ切り替えます。選んだモードは、次にアプリを開いた時も引き継がれます。

入力 画面下に 📷 1枚・📸 連写・🎤 音声・✏️ 手入力 を表示します。メニューの連続撮影・写真一括読み込みも使えます。
配送 配送先の入力ボタンを隠し、地図・配送リスト・配送順・ナビに集中できる画面にします。
別々の切り替えです: これは画面左上の配送・集荷コース選択とは別です。コースを変えずに、入力作業と配達作業だけを切り替えられます。

配送先の追加（伝票撮影・音声・手入力）

配送リストの見出しで 入力 を選んでから追加します。

🖼 写真をまとめて読み込み（おすすめ・大量向け）

  1. スマホのカメラアプリを普通に開いて、伝票を続けて全部撮影します（カメラは戻らないので最速です）。
  2. アプリに戻り、メニューの 🖼 写真をまとめて読み込み を押します。
  3. いま撮った伝票写真を全部まとめて選択します。
  4. 裏で1枚ずつ処理され、ピンが次々に立ちます。完了時に「追加◯件」のまとめが表示されます。

📷 伝票撮影（1枚ずつ・確認しながら）

  1. 📷 1枚 を押して伝票を撮影します。
  2. プレビューを確認して この向きで読み取り を押します（読み取りは裏で先に始まっているので、ほとんど待ちません）。
  3. 住所・備考を確認して 追加 を押します。
アプリ版（インストール型）なら: 📸 連写 を押すと、最初から連写用のアプリ内カメラが開きます。撮影した瞬間は白いフラッシュ・振動・短い音で分かります。

📸 連続撮影モード（数枚をその場で）

  1. 下の 📸 連写 を押します。アプリ版では連写用カメラが開きます。
  2. 白い枠の中に伝票を大きく入れ、シャッターが押せる状態になってから1枚ずつ撮影します。
  3. 写真は高めの解像度で裏から順に読み取られ、問題がなければそのままピンが立ちます。
  4. ❗要確認になった場合は連写が一旦止まり、その伝票の写真付き修正画面が自動で開きます。住所を確認して 保存 すると連写へ戻ります。
  5. 修正画面で 連写を終了 を押した場合は再開せず、❗要確認を配送リストに残します。
  6. 撮影を終える時はカメラ上部の 終了 を押します。

🎤 音声入力

  1. 🎤 音声 を押して、住所をゆっくり話します。
  2. 認識された住所を確認して追加します。

✏️ 手入力

  1. ✏️ 手入力 を押して、住所と備考を入力し 追加 を押します。
  2. 小舟町店・浜町店コースでは、町名・丁目をホイールで選び、番地・号を数字で入力する画面になります。番地・号は毎回空の状態から始まります。
  3. 連続入力 にチェックを入れると、追加後も画面が開いたままになり、続けて次を入力できます。終わるときは キャンセル で閉じます。

読み取りの自動サポート

伝票の読み取りには次の自動サポートが入っています。基本的に意識する必要はありません。

  • 向きの自動補正: 写真が横向き・逆さまでも、自動で回転して読み取り直します（数秒余分にかかります）。プレビューの回転ボタンは保険として残っています。
  • 宛名の取り込み: 伝票の宛名（◯◯様）は備考に「宛名: ◯◯」として自動で入ります。表札の確認に使えます。
  • 発送元の除外: お届け先と発送元が両方写っていても、発送元にはピンを立てません。判定が難しい場合は現在地に近い方をお届け先として採用します。
  • 郵便番号の活用: 郵便番号を住所検索に使い、町名の文字が一部欠けていても正しい町に解決されやすくしています。

❗要確認ピンと複数個口

❗要確認ピン（赤い枠）

追加したピンは自動で検査され、位置が怪しい場合は赤い枠と「❗要確認」バッジが付きます。理由も表示されます。

現在地・他のピン群から◯km配送先のピンが通常の配達エリアから極端に離れている可能性があります。「その他」の配送では遠方が正常なため、この距離警告は表示しません。
町名・番地が不一致読み取った住所と検索結果の正式住所が食い違っている。
郵便番号不一致伝票の〒と検索結果の〒が違う（似た町名の誤読の可能性大）。
概算位置住所検索の精度が低く、位置が仮。

「その他」の配送でも、概算位置、郵便番号・町名・番地の不一致、読み取り自信度低下による❗要確認は引き続き表示します。

連続撮影中は、❗要確認になった時点で撮影が止まり、伝票写真付きの修正画面が自動で開きます。写真と住所を見比べて 保存 すると❗が消え、連写へ戻ります。あとから確認する場合も、配送リストの ✎ で同じ写真を開けます。

直した位置は記憶されます: 📍 で正しい位置に動かすと、その住所の正しい位置として保存されます。次回以降、同じ住所を読み込むと最初から正しい位置にピンが立ちます（自動検索の結果で上書きされることはありません）。

📦 複数個口（同じ住所に複数の荷物）

  • 同じ住所の伝票をもう一度読み込むと、ピンは増えず荷物の個数が増えます（連続撮影では自動、1枚ずつの場合は確認が出ます）。
  • 個数が2以上のピンには紫の ×2 バッジ、リストには 📦 2個口 が表示されます。
  • 個数は ✎（編集）の 荷物の個数 から手で直せます。

時間指定の荷物

時間指定は手動で設定します（文字からの自動判定は誤作動が多かったため廃止しました）。

  1. リストの ✎（編集）で配送先を開きます。
  2. 時間指定 の欄から時間帯を選んで保存します。
  • 時間指定を付けたピンは、設定した色の枠で目立つ表示になります。
  • 時間帯の名前とピン色は、メニューの ⏰ 時間指定設定 で自由に編集できます。

集荷リストの取り込み

  1. メニューを開きます。
  2. 集荷コースを選択 を押します。
  3. コース、または 配送モード を選びます。
  4. このコースを開く を押します。
小舟町店を選んだ場合は 小舟町店スポット、浜町店 南/北を選んだ場合はそれぞれのスポット集荷も一緒に取り込みます。

集荷リストの下部は 定期集荷 と スポット集荷 のタブで切り替えられます。地図上にはどちらのピンも常に表示されます。

定期集荷はコースごとに端末へ保存されます。2回目以降は保存済みのピンを先に表示し、新規または住所が変わった集荷先だけ住所検索を行います。

スポット集荷

スポット集荷はエコ配システムから自動取得し、コースごとのスポット用シートに保存されます。

  • 7時台から17時台まで10分ごとに自動確認します。アプリを開いた時・コースを開いた時にもチェックが走ります。
  • 同じスポット集荷は同じIDで更新されるため、ピンが重複して増えません。
  • スポット集荷のピン番号は時間帯コードです。例: S09、S13、S16。
  • エコ配側でキャンセルになった集荷は完了済みと同じ扱いになり、担当者名の欄に キャンセル と表示されます。

スポット集荷の通知

新しいスポット集荷、またはキャンセルが見つかった時に、選択したコースだけスマホへ通知できます。

最初にアプリを追加する

iPhone Safariでアプリを開き、共有ボタンから ホーム画面に追加 を選びます。ホーム画面の RYS配送マップ テスト版 を開いて設定してください。iOS/iPadOS 16.4以降が必要です。
Android Chromeでアプリを開き、アプリをインストール または ホーム画面に追加 を選びます。

通知を有効にする

  1. ホーム画面に追加した RYS配送マップ テスト版 を開き、メニューの 通知設定 を開きます。
  2. 通知を受けるコースを1つ以上選びます。
  3. 有効化 / 更新 を押し、端末の通知許可を許可します。
ロック画面の通知には、コース名・時間帯コード・件数だけを表示します。会社名、住所、電話番号は表示しません。通知を押すと対象コースの集荷画面が開きます。

ピン表示と固定位置

通常ピン未完了の配送先・定期集荷です。
赤い枠のピン❗要確認。位置が怪しい可能性があります（詳細）。
紫の ×2 バッジ複数個口。数字が荷物の個数です。
スポット集荷ピン青色で、番号部分に S09 などの時間帯コードが出ます。完了すると緑色になります。
黄色いピン・? 付き住所検索の精度が低く、位置が仮の可能性があります。
色枠付きピン時間指定あり（詳細）。
2件 などのラベル同じ場所、または近い場所に複数のピンがあることを表します。

定期集荷の固定位置

定期集荷はスプレッドシートの lat、lng を優先してピンを立て、毎回の住所検索をしません。固定位置の変更には管理者PINが必要です。通常の配送先やスポット集荷の位置修正にPINは不要です。

完了・未完了の操作

  • リストの ✓ を押すと完了になります。↩ で未完了に戻ります。
  • 集荷モードでは、完了時に担当者名が必要です（メニューの 担当者名を設定）。
  • 通信が悪くて書き込めない場合は未同期の印が出て、回復後に自動再送されます。

ルート並べ替え

徒歩・自転車向け 近い順現在地から近い順に並べます。道路事情は考慮しません。
エリア相談 → 道路優先配送順（おすすめ・試験）車で配る時の基本機能です。GPSの青丸が地図に表示されてから使います。2件以上なら件数に関係なく使えます。町名ごとの順番を先に決め、件数に応じて最大20件ずつのブロックへ自動分割してMapboxの道路時間で計算します。次エリアへ移る道路時間も見て現在エリアの最後を決めるため、橋・線路・一方通行による遠回りを反映しながら、別エリアの荷物が途中に混ざらない順番にします。
エリア相談 → 直線距離で比較（道路未考慮）道路APIを使わず、選んだエリア順を崩さずに距離、折り返し、急な方向転換を比べます。徒歩での比較や道路順との比較用です。川・線路・一方通行は考慮しないため、車の配送順には「道路優先配送順」を使ってください。
車向け 道路優先ルートMapboxの道路ルートで一方通行等を考慮した順番を提案します。
Google車向け 道路優先ルートGoogle Routes APIで順番を提案します。結果を比べたい時に。

エリア相談画面では、最初に配りたいエリアを選ぶと残りが近い順に並びます。必要な場合は ↑ ↓ で順番を直してから、この順で計算 を押します。

並べ替え後は、現在地から1番までをオレンジ線、1番から2番までを青線で道路に沿って表示します。1番を完了にすると、現在地から2番までがオレンジ、2番から3番までが青へ自動で繰り上がります。道路ルートを取得できない時は、川や一方通行を越える直線を代わりに表示せず、警告を出します。その時は各ピンのナビを開いて確認してください。車向け道路優先ルートでは、この2本に加えて青い全体ルートが表示される場合があります。

翌日の集荷リセット

  • 集荷モードを開いた時に、前日以前の完了分を自動で未完了へ戻します。
  • 当日中に完了したものは、その日のうちは勝手に戻しません。
  • 手動で戻す場合はメニューの 集荷完了をリセット を使います。

番地の大きな表示と境界線

番地の大きな表示

  • 地図をズーム16.5以上にすると、建物・区画の番地が大きな文字で表示されます（全国対応・自動）。
  • 青い数字は国の住居表示データ（各戸の入口位置、例: 11-34）、黒い数字はMapboxの番地データです。
  • 一度表示したエリアは端末に保存され、次回から通信なしで即表示されます。走行中はGPS周辺を自動で先読みします。
  • メニューの 🏷 番地表示 でON/OFFできます。
  • 🔢 ハイフン番地 をOFFにすると、青い 11-34 のような数字だけを消します。黒い小さな番地はそのまま残ります。

番地境界線（法務省）

  • ズーム16以上で、土地の区画（筆）の境界線をオレンジの線で表示します。メニューの 📐 番地境界線(法務省) でON/OFF。
  • 法務省の地図データが整備済みの地域だけで表示されます。都市部の中心などデータの無い地域では出ません。

🩺 番地表示の状態（診断）

番地が出ない時は、メニューの 🩺 番地表示の状態 で状況を確認できます。🔍 筆界データ調査 で画面周辺のデータ有無を調べ、データがあれば 📍 データ地点へ移動 で確認できます。📋 コピー で診断内容をコピーして報告にも使えます。

高速起動とオフライン

  • アプリ本体は端末に保存され、2回目以降は一瞬で起動します。地図は前回終了した場所から始まります。
  • 圏外でもアプリは起動します。一度表示したエリアの番地も圏外で表示できます（地図タイルの新しい部分は電波が必要です）。
  • 新しいバージョンが出ると起動時にお知らせが出ます。手動で更新する場合はメニューの 🔄 アプリを更新 を押してください。

文字サイズ

メニューの Aa 文字サイズ から 小、標準、大 の3段階で変更できます。設定は端末ごとに保存されます。

困ったとき

ピンがまるで違う場所に立つ

多くの場合❗要確認（赤い枠）が付きます。✎ で開いて住所を直すか、📍 で正しい位置へ動かしてください。

一度直せばその住所の正しい位置として記憶され、次回からは最初から正しい位置に立ちます。

写真が読み取れない・向きが変

向きは自動補正されるため基本そのままで大丈夫です。読み取りに失敗した場合は写真が保持されたままプレビューに戻るので、その場で再試行できます。ピントが合っていない場合は撮り直してください。

番地の大きな表示が出ない

ズームを16.5以上にしてください（右上にズーム値が表示されています）。それでも出ない場合はメニューの 🩺 番地表示の状態 で状況を確認できます。

現在地が出ない

ブラウザの位置情報許可を確認してください。屋内ではGPS精度が落ちることがあります。

集荷の完了がスプレッドシートに反映されない

担当者名が未設定の場合は先に設定してください。通信が悪い場合は未同期として保存され、回復後に自動再送されます。

スポット集荷が表示されない

集荷コースをもう一度開いてください。短時間に連続して開いた場合は、少し待ってから反映されることがあります。

通知が届かない

メニューの 通知設定 で状態と選択コースを確認してください。ホーム画面のアプリから開き直し、端末の通知許可も確認してください。

車用ルートが失敗する

配送先が25点以上ある場合は24件以内に分けて検索してください。それでも失敗する場合は管理者に連絡してください。

画面が古いままに見える

メニューの 🔄 アプリを更新 を押してください。
