🎮 遊び方

ステージ1: 住所ピン当て(5問)
実在の住所が出題されます。地図をタップして「ここだ!」と思う場所にピンを刺そう。正解に近いほど高得点。

ステージ2: ルート組み立て
当てた5つの配達先を、お店から最短で回る順番を考えよう。AIが計算した最適ルートと勝負!

段位認定
総合スコアで「見習い → 配達士 → 達人 → 伝説の配達員」の段位を認定。デイリーチャレンジは毎日全員に同じ問題が出ます。

📚 遊ぶと身につくこと

日本独特の住所のしくみ(丁目・番・号)、日本橋エリアの地理、そして配送の現場で実際に使われている「ルート最適化」の考え方が、遊びながら自然に学べます。
